Quantified Expression

Quantified expressions can be used to extend assertion conditions. There are two quantifiers available to use: the exists quantifier and the for all quantifier.

Exists

The exists quantifier can be used to check if there is at least one value for an integer variable for which a condition is true.

For All

The for all quantifier can be used to check if all values for an integer variable match a condition.

Range

Both quantifiers can be supplemented with a range in which the value of the variable has to be in to match the condition. If no range is given, a verification tool will be used to determine if the condition is true or false.

Examples

int x = 4;
assert exists y(3,6) y == x;

assert forall x(0,10) x/2 == (x+1)/2;

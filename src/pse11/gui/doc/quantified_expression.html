Quantified Expression

Quantified expressions can be used to extend specifications. There are two quantifiers available to use: the exists quantifier and the for all quantifier.

Every quantified expression can be substitued by an expression which must not contain a function call.

Exists

The exists quantifier can be used to check if there is at least one value for an integer variable for which a condition is true.

For All

The for all quantifier can be used to check if all values for an integer variable match a condition.

Range

Both quantifiers can optionally be supplemented with a range in which the value of the variable has to be in to match the condition. If no range is given, an implicit assertion will fail during runtime.

Syntax

quantifier identifier( expression, expression ) quantified expression

where quantifier is one of exists or forall.

Example

exists j(0,15) j==5 forall s() s > 0

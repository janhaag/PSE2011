Quantified Expression

Quantified expressions can be used to extend assertion conditions. There are two quantifiers available to use: the exists quantifier and the for all quantifier.

Every quantified expression can be substituted by an expression.

Exists

The exists quantifier can be used to check if there is at least one value for an integer variable for which a condition is true.

For All

The for all quantifier can be used to check if all values for an integer variable match a condition.

Range

Both quantifiers can optionally be supplemented with a range in which the value of the variable has to be in to match the condition. If no range is given, a forall quantifier is always evaluated to false and an exists quantifier is always evaluated to true during runtime.

Syntax

quantifier identifier( expression, expression )*

where quantifier is one of exists or forall.

Example

assert exists j(0,15) j==s;

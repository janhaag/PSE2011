Functions

Functions are a sequence of statements which can be executed separately when provided with specified parameters and returning a result.

Function Declaration

Functions are declared by the type of its return value, the function name, a parameter list and the function body.

Syntax

type identifier ( [type identifier [, type identifier]* ] ) [assume] { [statement]* returnexpression; } [ensure]

Example

bool f (int x) { return x > 5; }

Main Function

Every runnable program must have a main function, which is a special function without a return type. Program execution will start at the beginning of the main function.

main ( [type identifier [, type identifier]* ] ) assume { [statement]* } [ensure]

Example

main() { int x = 4; if(x < 0) { x = -x; } }

Function Call

Functions can be called by their name and the parameters in round brackets, which will execute the function with the specified parameters and return a value. Function calls are expressions.

Syntax

identifier ( [expression [, expression]* ] )

Example

f(x+3);

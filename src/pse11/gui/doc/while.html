While Statement

A while statement is a control flow statement. The statement block following the while-statement is only executed if a specified condition is true. If the condition is true after leaving that statement block, the statement block is entered again. If the condition is false when encountering a while-statement during execution or after leaving the statement block after a while-statement, the execution continues after that statement block. A while-loop can be extended by the assertion statements invariant and ensure, which are optional. See the article about assertions for further information.

Syntax

while ( expression ) invariant { statement* } ensure

Example

int i = 0; int s = 0; while(i < 5) { i = i + 1; s = s + i; }

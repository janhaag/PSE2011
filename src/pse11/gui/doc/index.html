Introduction

Loading and Saving

Any file with the extension 'txt' or 'wp' can be read and its content be placed into the editor with 'File' - 'Load'. The editor's content can be saved into a file with 'File' - 'Save'.

Interpreting a Program

A program with correct syntax can be interpreted, thus executing its statements. When interpreting a program, you can either run the program, which will only stop when a runtime error has occured, or execute the program in single-step mode, which will cause the interpretation to stop after every executed statement.

Verifying a Program

A program with correct syntax can be verified. This is done using the theorem prover Z3 developed by Microsoft. A verification can result in the program being verified as correct, as incorrect or unknown. The verifier tries to use axioms in the program to conclude all assertions.

Syntax in this Documentation

The syntax help has additional annotations using boxes:

  • optional means that the syntax element in the box is optional
  • multiple and optional* means that the syntax element in the box is optional and can be repeated any number of times
  • syntax element means that the syntax element in the box is to be replaced by the syntax of another syntax element
  • text without any further formatting means that the text is to be used without modification

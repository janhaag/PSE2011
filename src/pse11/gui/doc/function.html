Functions

Functions are a sequence of statements which can be executed separately when provided with specified parameters and returning a result.

Function Declaration

Functions are declared by the type of its return value, the function name, a parameter list and the function body:

return_type function_name(paramter1, parameter2, ...) assume-statement {
function_body
} ensure-statement

Both the assume-statement and ensure-statement are optional. See the article about assertions for further information.

Calling functions

Functions can be called by their name and the parameters in round brackets, which will execute the function with the specified parameters and return a value.

Return Statement

Every function (with the exception of the main function) returns a value specified by its return type. To do this, use the return keyword followed by the value to be returned. There can only be one return statement per function and it has to be the last statement in a function.

Main Function

Every runnable program must have a main function, which is a special function without a return type. Program execution will start at the beginning of the main function.

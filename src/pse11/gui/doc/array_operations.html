Array operations

When two arrays have the same size in each dimension and the same base type, the following operations are available:

len (length)
length(a1)
Returns the length of the next dimension of a1 (e.g. for a 2-dimensional array a = array[1][2], length(a) returns 1 and length(a[0]) returns 2.
== (equals)
a1 == a2
Returns true if all components of the two arrays a1 and a2 are equal for every index.
!= (not equals)
a1 != a2
Returns true if there is one index for which the components of a1 and a2 are not equal.

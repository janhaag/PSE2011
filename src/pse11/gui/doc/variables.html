Variable Declaration

Variables are names that can be used to save certain values. Variables can either hold an integer, a boolean (true or false) or an array of values.

Integer Variables

Integer variables are declared using the int keyword. These variables can hold a signed integer of any size.

Integer variables are initialized with a default value of 0 if no value is given.

Syntax

int identifier = expression;

Example

int x; int y = -3;

Boolean Variables

Boolean variables are declared using the bool keyword. These variables can hold one of the values true or false.

Boolean variables are initialized with a default value of false if no value is given.

Syntax

bool identifier = expression;

Example

bool r; bool s = true;

Arrays

Arrays can hold many values of a single type. Arrays can be declared by adding square brackets ([]) to the type of a variable declaration to specify the dimension of the array and specifying the size of each dimension by using the array keyword followed by square brackets which contain the size of each dimension. Arrays will count their components starting with 0. To access a component of an array, use the name of the array followed by one pair of square brackets containing the index for each dimension. When accessing array components, there will be an implicit check if the indices provided are inside of the array's bounds. The length of an array can be obtained by using the len function and providing the name of the array as parameter.

Every component of an array is initialized to the default value of the component's type, i.e. 0 for integer variables and false for boolean variables.

Syntax

type[][]* identifier = array[expression][expression]*;

where type is one of int or bool.

Example

int[] a = array[4]; bool[][] b = array[6][4];

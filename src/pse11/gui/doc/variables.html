Variables

Variables are names that can be used to save certain values. Variables can either hold an integer, a boolean (true or false) or an array of values.

Integer Variables

Integer variables are declared using the int keyword. These variables can hold a signed integer of any size.

Boolean Variables

Boolean variables are declared using the bool keyword. These variables can hold one of the values true or false.

Arrays

Arrays can hold many values of a single type. Arrays can be declared by adding square brackets ([]) to the type of a variable declaration to specify the dimension of the array and specifying the size of each dimension by using the array keyword followed by square brackets which contain the size of each dimension.

Examples

int x;
int y = 5;

bool a = true;
bool b;

int[] s = array[5];
bool[][] t = bool[4][6];

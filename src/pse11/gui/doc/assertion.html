Assertion

An assertion is a predicate that describes the current state of a program.

If an assertion is encountered while running a program, the condition of that assertion will be checked. If the condition is false, the program execution will halt; otherwise, the program execution will continue. The condition of an assertion is an expression that is evaluated to a boolean, allowing the usage of quantified expressions.

Assertions can additionally be added to functions and while-loops with the ensure keyword, resulting in a check once the execution of the function body or the loop body are finished. While-loops can contain an invariant, which is an assertion that is checked every time the loop body execution is finished. Functions can contain assumes, which are checked on function body entry.

A program can only be proven correct if all assertions are true.

Axioms

Axioms are created by using the axiom keyword. Axioms are special assertions, as they are not checked at runtime and will not be proved, but instead be used to prove other assertions in the program.

Syntax

assert condition;

type function(parameters) assume { condition; ... } {
function body
} ensure { condition; ... }

while(loop condition) invariant { condition; ... } {
function body
} ensure{ condition; ... }

Examples

int function(int x) assume x > 0 {
int y = x + 1;
} ensure y > 0;

int x = 4;
assert exists y(3,6) y == x;

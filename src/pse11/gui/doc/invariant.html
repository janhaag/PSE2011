Invariant

An invariant is an assertion which is specific to while statements. The condition of an invariant is checked once the while statement is encountered (even if the condition is false) and every time the loop body execution is finished. Regarding the interpretation or validation, see the article about specifications.

Syntax

invariant quantified expression;
invariant { quantified expression; quantified expression;* }

Example

int i = 0; int s = 0; while(i < 5) invariant s == i*(i+1)/2; { i = i + 1; s = s + i; }

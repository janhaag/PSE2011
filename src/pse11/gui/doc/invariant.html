Invariant

An invariant is an assertion which is specific to while statements. The condition of an invariant is checked every time the loop body execution of a while statement is finished.

Syntax

invariant quantified expression;
invariant { quantified expression; quantified expression;* }

Example

int i = 0; int s = 0; while(i < 5) invariant s == i*(i+1)/2; { i = i + 1; s = s + i; }

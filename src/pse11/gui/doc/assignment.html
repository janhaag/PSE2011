Assignment

Assignments are statements that assign a new value to a variable. A variable must be declared before it can be assigned a value.

Syntax

identifier = expression;

When using an array variable, you have to specify the index of each dimension:

identifier[expression] [expression]* = expression;

Note that there is an implicit check if array indices are valid (greater than or equal to zero and lower than the length of the dimension) for an array when assigning values to it, so the interpreter will stop twice for array assignments in single step mode.

Also note that an array assignment is only valid on completely indexed arrays, i.e. every dimension of an array has to be indexed. Assignments to non-completely indexed arrays (e.g. 'a = ...', where a is an int[]) are not possible.

Example

int x; x = 5; bool[] a = array[2]; a[1] = true;

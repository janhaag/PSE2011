Integer operations

The following operations on integers are available in the while-language:

+ (addition)
int1 + int2
Returns the sum of the two integers.
- (subtraction)
int1 - int2
Returns int1 minus int2.
* (multiplication)
int1 * int2
Returns the product of the two integers.
/ (division)
int1 / int2
Returns int1 divided by int2, rounded towards zero. The result of division by zero is defined as zero, i.e. int1 / 0 = 0.
% (modulo)
int1 % int2
Returns the remainder of the integer division of int1 divided by int2. Note that int1 % 0 = int1.
- (negation)
-int1
Returns the negation of the integer.
== (equals)
int1 == int2
Returns true if both integers have the same value; returns false otherwise.
!= (not equals)
int1 != int2
Returns false if both integers have the same value; returns true otherwise.
<= (less or equal)
int1 <= int2
Returns true if int1 is less than or equal to int2; returns false otherwise.
< (less)
int1 < int2
Returns true if int1 is less than int2; returns false otherwise.
>= (greater or equal)
int1 >= int2
Returns true if int1 is greater than or equal to int2; returns false otherwise.
> (greater)
int1 > int2
Returns true if int1 is greater than int2; returns false otherwise.

Notes

When running a program in single step mode, the interpreter stops twice at each division. This is due to an implicit check of a division by zero error.

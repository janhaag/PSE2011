Integer operations

The following operations on integers are available in the while-language:

+ (addition)
int1 + int2
Returns the sum of the two integers.
- (subtraction)
int1 - int2
Returns int1 minus int2.
* (multiplication)
int1 * int2
Returns the product of the two integers.
/ (division)
int1 / int2
Returns int1 divided by int2. It is implicitly checked whether int2 (the divisor) is zero; execution will stop if possible, the result will be zero otherwise (e.g. for bounds of quantified expressions). See below for an exact definition.
% (modulo)
int1 % int2
Returns the remainder of the division of int1 divided by int2. It is implicitly checked whether int2 (the divisor) is zero; execution will stop if possible, the result will be int1 otherwise (the result will be zero otherwise (e.g. for bounds of quantified expressions). See below for an exact definition.
- (negation)
-int1
Returns the negation of the integer.
== (equals)
int1 == int2
Returns true if both integers have the same value; returns false otherwise.
!= (not equals)
int1 != int2
Returns false if both integers have the same value; returns true otherwise.
<= (less or equal)
int1 <= int2
Returns true if int1 is less than or equal to int2; returns false otherwise.
< (less)
int1 < int2
Returns true if int1 is less than int2; returns false otherwise.
>= (greater or equal)
int1 >= int2
Returns true if int1 is greater than or equal to int2; returns false otherwise.
> (greater)
int1 > int2
Returns true if int1 is greater than int2; returns false otherwise.

Notes

When running a program in single step mode, the interpreter stops twice at each division and modulo operator. This is due to an implicit check of a division by zero error.

For integers a and b, b not equal to zero, division and modulo is defined to satisfy the formulas
a = b * (a / b) + a % b
and 0 <= a % b <= |n| - 1
where |n| is the absolute value of n.

For an integer a and a positive integer b, a / b equals the floor of the rational number a / b.
For an integer a and a negative integer b, a / b equals the ceiling of the rational number a / b.

For integers a and b not equal to zero, a % b equals a - (a / b) * b.

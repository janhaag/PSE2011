Assume

Functions (including the main function) can be extended by using assumes. Assumes are special assertions that are checked once on function body entry.

Syntax

assume quantified expression;
assume { quantified expression; [quantified expression;]* }

Example

bool f (int x) assume x > 3; { return x > 5; } int g (int x) assume {x > 3; x < 10;} { return x * 2; }

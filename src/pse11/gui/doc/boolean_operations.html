Boolean operations

The following operations on booleans are available in the while-language:

| (logical or)
bool1 | bool2
Returns true if at least one of the booleans is true; returns false otherwise.
& (logical and)
bool1 & bool2 (logical and)
Returns true if both booleans are true; returns false otherwise.
! (logical not)
!bool1
Returns the true if the boolean is false; returns false if the boolean is true.
== (equals)
bool1 == bool2
Returns true if both booleans have the same value; returns false otherwise.
!= (not equals)
bool1 != bool2
Returns false if both booleans have the same value; returns true otherwise.
